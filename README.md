# Blinkit_PowerBI
This project presents an interactive Blinkit Sales Dashboard built using Power BI to analyze sales performance, customer behavior, and outlet characteristics. The dashboard provides deep insights into product categories, outlet types, and regional performance to support data-driven decision-making.

**Objectives:**
Analyze overall sales performance and key KPIs,
understand the impact of outlet size, type, and location on sales,
identify top-performing product categories,
evaluate customer preferences based on fat content and item types,
generate actionable business insights.

**Key Insights:**
Low Fat products contribute significantly to total sales,
tier 3 cities generate the highest revenue among all locations,
supermarket Type 1 outlets dominate in overall sales,
medium-sized outlets contribute a major portion of revenue,
categories like Fruits, Snacks, and Household items perform the best.

**Results & Conclusion:**
The dashboard effectively transforms raw retail data into meaningful insights. It helps identify key revenue drivers, optimize product offerings, and evaluate outlet performance, enabling better strategic decisions.
